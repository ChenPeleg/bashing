# get virtual env in windows with ssl and poetry

## install pip
    
    ```bash
    python -m pip install --upgrade pip
    ```


### Trouble shooting

if you get the error:

    `Package 'python3-pip' has no installation candidate`

run:

    ```bash
    sudo apt-get update
    ```

And then


    ```bash
    sudo apt-get install python3-pip
    ```
